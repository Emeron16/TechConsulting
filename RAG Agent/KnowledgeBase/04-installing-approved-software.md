# Installing Approved Software via the TC App Store

TechConsulting maintains a curated catalog of pre-approved, security-reviewed applications called the **TC App Store**. Employees can self-install anything in this catalog without filing a ticket. Software not in the catalog requires a request and a security review.

## Currently approved applications (partial list)

Zoom, Slack, Adobe Acrobat Reader DC, 7-Zip, Google Chrome, Microsoft Visual Studio Code, Notepad++, Figma, and Postman. The full, current list is always available in the TC App Store itself, since this document may lag behind additions.

## Steps (Windows)

1. Open the **Company Portal** app from the Start menu (this is the TC App Store).
2. Search for the application by name.
3. Click **Install**. No admin password is required — approved apps are pre-authorized to install under your standard user account.
4. Once installed, the app will appear in your Start menu.

## Requesting a new application

If the software you need isn't in the catalog, submit a request through the TC Helpdesk Portal with the application name, vendor, and business justification. Requests are reviewed by IT Security, typically within 3–5 business days. Do not download and install software directly from the internet outside the App Store — this violates the TechConsulting security policy and unapproved installers will be automatically blocked on managed devices.
