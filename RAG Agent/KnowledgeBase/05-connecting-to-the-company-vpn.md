# Connecting to the Company VPN (TC SecureConnect)

TC SecureConnect is TechConsulting's VPN client, required for accessing internal file servers, print servers, and internal web tools whenever you're off the office network.

## Installing the client

TC SecureConnect is pre-installed on all company-issued laptops. If it's missing, install it from the TC App Store (see "Installing Approved Software").

## Steps to connect

1. Open **TC SecureConnect** from the system tray or Start menu.
2. In the server field, confirm it shows `vpn.techconsulting.com` (this is pre-filled by default).
3. Click **Connect**.
4. Enter your TechConsulting network username and password.
5. Approve the push notification sent to your Microsoft Authenticator app (VPN connections require MFA — see "Enrolling in Multi-Factor Authentication").
6. Once connected, the system tray icon turns green and a "Connected" notification appears.

## Troubleshooting

If the connection fails immediately, check your internet connection first — VPN requires an active internet connection to reach the VPN gateway. If you don't receive an MFA push, open Microsoft Authenticator manually and check for a pending request, or reinstall the Authenticator app if it isn't receiving notifications. If problems persist for more than 10 minutes, check status.techconsulting.com for a known outage before contacting IT.
